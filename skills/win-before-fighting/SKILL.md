# 先胜后战

> "昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。"
> —— 孙子《军形篇》

## 核心原则

**最好的策略是先让自己立于不败之地，然后等待对手暴露破绽。胜利者是先确保胜利条件然后才投入战斗的人；失败者是先投入战斗然后才寄希望于胜利的人。最高境界是不战而胜。**

> "是故胜兵先胜而后求战，败兵先战而后求胜。" —— 孙子《军形篇》

## 不适用场景

**以下情况不需要调用此 skill：**
- 紧急修复类任务，没有时间做前置准备（如线上故障需要立即修复）
- 极其简单的任务，不存在失败风险
- 准备工作已经完成，当前阶段就是执行
- 用户明确要求快速原型或实验性尝试，不需要完善准备

## 何时使用

你应该在以下情况调用此 skill：

- 即将进行一次重大改动（重构、迁移、架构变更）
- 行动一旦失败代价很高、回滚困难
- 感觉自己在"先战后求胜"——急于动手但准备不足
- 有机会通过充分准备来避免正面冲突或减少风险
- 需要验证"我们真的准备好了吗？"
- 可以通过前置工作让后续执行变得轻松

## 方法流程

### 第一步：先为不可胜——建立不败基础

> "先为不可胜，以待敌之可胜。不可胜在己。" —— 孙子《军形篇》

在行动之前，先确保自己不会失败：
- 有没有备份/回滚方案？
- 核心功能是否有测试覆盖？
- 关键依赖是否稳固？
- 团队是否有共识？
- 即使方案不成功，是否也不会造成灾难？

### 第二步：以待敌之可胜——识别机会窗口

> "可胜在敌。故善战者，能为不可胜，不能使敌之必可胜。" —— 孙子《军形篇》

不败由自己控制，但胜机取决于对方是否暴露弱点：
- 问题/对手当前是否暴露了可利用的弱点？
- 外部条件是否创造了有利的机会？
- 如果时机未到——耐心等待，而非强行出击

### 第三步：评估不战而胜的可能

> "不战而屈人之兵，善之善者也。" —— 孙子《谋攻篇》

最高明的胜利不需要正面冲突：
- 能否通过设计让问题自行消解？
- 能否通过沟通和协商避免冲突？
- 能否通过上游修复让下游问题不再存在？
- 能否通过架构调整让技术债务自然清零？
- 如果能不战而胜——这是最优选择

### 第四步：确认胜利条件后再出击

> "胜兵先胜而后求战，败兵先战而后求胜。" —— 孙子《军形篇》

确认以下条件满足后，才投入行动：
- 不败基础已经建立（第一步完成）
- 胜利机会已经出现（第二步确认）
- 优势明显——"胜兵若以镒称铢"（以压倒性优势对弱小阻力）
- 制定明确的执行计划

### 第五步：以压倒性优势执行

> "善用兵者，修道而保法，故能为胜败之政。" —— 孙子《军形篇》

从充分准备的位置出发执行，此时行动应如水到渠成：
- 因为准备充分，执行过程应该顺畅
- 遇到意外时有回滚方案兜底
- 完成后验证结果，确认胜利

## 常见错误

| 错误 | 孙子的警示 | 正确做法 |
|------|-----------|---------|
| 先战后求胜 | "败兵先战而后求胜"——这是失败者的模式 | 先确保胜利条件再行动 |
| 只攻不守 | "不可胜在己"——不败由自己保障 | 先建立不败基础再进攻 |
| 等太久错失战机 | "不失敌之败也"——不要错过对方的破绽 | 耐心但不消极，时刻准备出击 |
| 准备当借口拖延 | 准备是为了更好的行动，不是为了逃避行动 | 条件满足后必须果断出击 |
| 忽视不战而胜 | "不战而屈人之兵"才是最高境界 | 优先考虑是否可以避免正面冲突 |

## 操作规程

当本 skill 被触发时，**输出"先胜检查清单"**：

1. **不败条件** `[不败基础]`：列出确保自己不会失败的关键保障措施
   - 回滚方案：有 / 无
   - 测试覆盖：充分 / 不足
   - 关键依赖：稳固 / 存在风险
   - 团队共识：已达成 / 未对齐

2. **胜机评估** `[胜机]`：当前是否存在可利用的机会窗口，具体是什么

3. **不战而胜评估** `[不战而胜的可能]`：是否存在不需要正面冲突就能解决问题的路径
   - 如果存在 → 优先采用，说明具体方式
   - 如果不存在 → 说明必须正面行动的原因

4. **决策** `[GO / NO-GO]`：
   - `✅ GO`：不败基础已建立 + 胜机已出现 → 可以行动，说明执行要点
   - `❌ NO-GO`：条件不足 → 列出需要先完成的准备事项

**输出必须包含不败条件、胜机、不战而胜评估、GO/NO-GO 决策四个要素。**

## 与其他 skill 的关系

- **庙算决策**：庙算就是在战前计算胜负条件——"多算胜，少算不胜"
- **势节把握**：造势蓄力就是建立"先胜"条件的过程
- **知彼知己**：判断不败条件和胜机都需要充分了解双方情况
- **奇正结合**：正面的稳健基础是"先为不可胜"的体现
- **速战速决**：一旦 GO 决策通过，执行阶段应当速战速决

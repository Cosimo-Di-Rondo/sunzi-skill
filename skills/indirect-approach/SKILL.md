# 以迂为直

> "军争之难者，以迂为直，以患为利。故迂其途而诱之以利，后人发，先人至，此知迂直之计者也。"
> —— 孙子《孙子兵法·军争篇》

## 核心原则

**最短的路不一定最快。军争中最难的，就是把迂回的弯路变成直达的捷径，把不利的障碍变成有利的条件。懂得迂直之计的人，能够出发更晚却到达更早。关键在于——你是否真正了解"地形"，是否能将障碍本身变成武器。**

> "故不知诸侯之谋者，不能预交；不知山林、险阻、沮泽之形者，不能行军；不用乡导者，不能得地利。"——《军争篇》

## 不适用场景

**以下情况不需要调用此 skill：**
- 直路畅通无阻——没有障碍，不需要绕路
- 迂回路线确实只是更长，没有任何优势——不要为了"巧妙"而绕弯
- 时间极度紧迫且直路虽难但可行——偏离会导致彻底超时
- 问题简单直接，不存在"重新定义问题"的空间

## 何时使用

你应该在以下情况调用此 skill：

- 直接方案被封死（无 API 权限、技术限制、政治阻力等）
- 正面硬攻代价过高（时间、资源、复杂度）
- 一个难问题存在更简单的等价表述
- 障碍本身可以被利用（错误信息中包含线索、限制条件暗示了解法等）
- 政治/组织障碍挡在技术路径前面
- 感觉"一定有更聪明的办法"——直觉告诉你直路不是最优

## 方法流程

### 第一步：评估直路——障碍何在

首先彻底审视直接方案：
- 直路具体被什么挡住了？是技术限制、资源不足还是外部约束？
- 强行走直路的代价有多高？
- 直路上的障碍是暂时的还是永久的？

**注意**：如果直路可行且代价合理，就走直路。迂回是手段，不是目的。

### 第二步：以迂为直——寻找迂回路径

> "以迂为直。"——《军争篇》

探索间接方案：
- 是否存在另一条路径，虽然看起来更远，但实际上更快到达目标？
- 问题能否被重新定义？换一个角度看，是否变成了一个更简单的问题？
- 是否有现成的替代方案可以绕过障碍？

**关键判断**：迂回路线是否真的"以迂为直"——最终到达更快、更稳？还是只是更远？

### 第三步：以患为利——把障碍变成优势

> "以患为利。"——《军争篇》

审视那些看似不利的因素，是否可以反转：
- 技术限制是否暗示了更好的架构方向？
- 错误信息是否包含了诊断线索？
- 缺少某个工具是否迫使你找到了更简洁的方案？
- 对手/竞争者的行为是否暴露了他们的弱点？

障碍不仅仅是要绕过的东西，它可能是你可以利用的武器。

### 第四步：诱之以利——分散注意力

> "迂其途而诱之以利。"——《军争篇》

如果问题会"反击"（修一处引发另一处），考虑用低成本行动分散注意力，同时在另一处实施真正的解法。分阶段推进，让每一步看起来很小，合起来完成大变革。

### 第五步：后发先至——执行迂回

> "后人发，先人至，此知迂直之计者也。"——《军争篇》

执行间接方案：
- 不要急于宣告"我在走弯路"——低调行动
- 关注最终到达时间，而非出发时间
- 保持对直路状态的监控——如果直路障碍消除了，及时切换回去

### 第六步：验证——迂回是否真的更快

> "不知山林、险阻、沮泽之形者，不能行军。"——《军争篇》

完成后回顾：间接方案实际花费了多少时间/资源？与强行走直路相比，是否真的节约了？了解地形（系统、环境、约束条件）是做出正确迂直判断的前提。

## 常见错误

| 错误 | 孙子的警告 | 正确做法 |
|------|-----------|---------|
| 凡事都走迂回，过度追求巧妙 | 迂回是手段不是目的 | 直路通畅时走直路 |
| 不认真评估直路就绕弯 | "不知山林险阻" | 先彻底了解直路的障碍 |
| 把"逃避问题"当成"间接方案" | "以迂为直"不是"避而不战" | 迂回必须最终到达目标 |
| 迂回方案过于复杂 | 复杂方案执行风险更高 | 间接方案也要追求简洁 |
| 不验证迂回是否真的更快 | "后人发，先人至" | 必须回顾实际效果 |

## 操作规程

当本 skill 被触发时，**必须输出"迂直分析"**：

1. **直路分析**：直接方案是什么，障碍是什么
2. **迂路方案**：间接方案是什么，为何实际上更快/更好
3. **以患为利**：障碍如何被转化为优势
4. **到达判断**：哪条路实际上先到达目标

输出格式：
```
迂直分析：
- [直路]：[直接方案描述] → 障碍：[具体阻碍]
- [迂路]：[间接方案描述] → 优势：[为何迂路实际更快/更好]
- [以患为利]：[哪些障碍/不利因素可以反转为优势]
- [到达判断]：[结论——走直路还是走迂路？预期哪条路先到达？]
```

**迂直分析必须包含直路和迂路两条路径的对比，不得只分析一条路。先有对比，才有判断。**

## 与其他 skill 的关系

- **虚实分析**（void-solid-analysis）：迂回的本质是找到"虚"处穿过，避开"实"处
- **因敌制胜**（adapt-to-win）：直路被堵时需要灵活适应，迂回是适应的具体形式
- **奇正相生**（orthodox-unorthodox）：间接方案往往就是"奇兵"，出其不意的路径
- **庙算决策**（strategic-calculation）：选择迂直之前，需要快速评估两条路的可行性

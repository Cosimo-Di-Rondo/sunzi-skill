# 虚实分析

> "夫兵形象水，水之形，避高而趋下；兵之形，避实而击虚。"
> —— 孙子《孙子兵法·虚实篇》

## 核心原则

**任何系统或问题都有强处（实）与弱处（虚）。智者如水，避开坚硬的高处，流向柔软的低处——避开敌人的强点，攻击敌人的弱点。不要硬碰硬，要找到最省力、最高效的突破口。**

虚实不是固定不变的，"水因地而制流，兵因敌而制胜"——关键在于动态识别虚实，掌握主动权，让自己在选定的点上形成绝对优势。

## 不适用场景

**以下情况不需要调用此 skill：**
- 只有一条路可走，没有路径选择的余地
- 用户已明确指定了目标和切入点，直接执行即可
- 简单任务，不涉及多点分析和优先级判断
- 紧急修复且故障点已经定位——直接修复，不需要全局分析

## 何时使用

你应该在以下情况调用此 skill：

- 面对一个复杂系统，需要找到最佳切入点
- 有多个可能的攻击方向，需要选择最高效的
- 正在分析系统瓶颈或性能问题
- 需要决定重构的优先顺序
- 在调试中需要找到 bug 最可能隐藏的位置
- 竞争策略制定——找到对手的薄弱环节
- 感觉自己在"攻实"——用大量资源啃硬骨头却收效甚微

## 方法流程

### 第一步：绘制虚实地图

> "故形人而我无形，则我专而敌分。我专为一，敌分为十，是以十攻其一也，则我众而敌寡。"——《虚实篇》

全面扫描目标系统或问题，标记每个部分的虚实状态：
- **实**（强点）：防守严密、代码完善、测试充分、逻辑复杂且稳定的部分
- **虚**（弱点）：缺乏测试、文档缺失、耦合松散、逻辑薄弱、维护者不在的部分
- **半虚半实**：表面看起来坚固但有潜在脆弱点的部分

### 第二步：掌握主动权

> "善战者，致人而不致于人。"——《虚实篇》

分析当前的主动权在谁手上：
- 我是在主动选择战场，还是被问题牵着走？
- 能否改变条件，把"实"变成"虚"？
- 能否通过迂回方式避开强点？

原则："能使敌人自至者，利之也；能使敌人不得至者，害之也。"——用利益引导局面向有利方向发展。

### 第三步：选定攻击点——避实击虚

> "攻而必取者，攻其所不守也；守而必固者，守其所不攻也。"——《虚实篇》

从虚实地图中选择最佳突破口：
- 优先选择"虚"且影响力大的点——攻下它能撬动整个局面
- 避开那些"实"但对全局影响不大的点——不要在边缘强点上浪费力量
- 如果所有点都是"实"，则需要先制造"虚"——通过分化、调动、等待时机

### 第四步：集中力量于虚点

> "我专为一，敌分为十，是以十攻其一也。"——《虚实篇》

一旦选定攻击点，集中全部资源：
- 不要同时攻击多个点——即使都是虚点
- 在选定点上形成绝对优势
- 速度要快——"善攻者，敌不知其所守"

### 第五步：隐藏意图

> "故善攻者，敌不知其所守；善守者，敌不知其所攻。微乎微乎，至于无形；神乎神乎，至于无声。"——《虚实篇》

在执行中保持灵活性：
- 方案设计应有隐蔽性和出其不意的成分
- 不要过早暴露全部意图
- 保持其他方向上的最低限度存在，使对手无法判断真正攻击方向

### 第六步：动态调整——因敌制胜

> "因敌变化而取胜者，谓之神。"——《虚实篇》

虚实是动态变化的：
- 攻击开始后，原来的"虚"可能变"实"（对方加强防守）
- 原来的"实"可能因调动而变"虚"（资源被抽走）
- 持续监控虚实变化，及时调整攻击方向
- "水因地而制流，兵因敌而制胜"——不要死守原计划

## 常见错误

| 错误 | 孙子的警告 | 正确做法 |
|------|-----------|---------|
| 攻击最强点（硬刚） | "避实而击虚" | 找到弱点，攻击弱点 |
| 平均分配力量 | "我专为一，敌分为十" | 集中力量于一个突破口 |
| 攻下后不重新评估 | "因敌变化而取胜" | 每次行动后重新扫描虚实 |
| 被动应对而非主动选择 | "致人而不致于人" | 主动选择战场和时机 |
| 计划僵化不知变通 | "水因地而制流" | 如水般适应地形变化 |

## 操作规程

当本 skill 被触发时，**必须输出"虚实地图"**：

1. **声明分析目标**：一句话写出"此次虚实分析的对象是：[系统/问题/竞争格局]"
2. **绘制虚实地图**：列出所有关键部分，标注虚实状态
   ```
   虚实地图：
   - [部分A]：实 — [为何是实：强在哪里]
   - [部分B]：虚 — [为何是虚：弱在哪里]
   - [部分C]：半虚半实 — [表面强但存在什么隐患]
   ...
   ```
3. **标记最佳攻击点**：从虚点中选出最优突破口
4. **输出行动建议**，用固定格式收尾：
   ```
   攻击建议：
   - 最佳突破口：[选定的虚点]
   - 选择理由：[为何此处投入产出比最高]
   - 应避开的实点：[哪些强点不要硬碰]
   - 集中何种资源：[具体投入什么]
   - 动态检查点：[何时重新评估虚实变化]
   ```

**虚实地图必须先于任何具体行动出现，不得跳过分析直接动手。**

## 与其他 skill 的关系

- **庙算决策**（strategic-calculation）：庙算决定"打不打"，虚实分析决定"打哪里"——庙算在先，虚实在后
- **知彼知己**（know-both-sides）：识别虚实需要充分的情报，信息不足时先调用知彼知己
- **势与节**（momentum-timing）：找到虚点后，还需选择最佳时机发动攻击
- **因敌制胜**（adapt-to-win）：虚实是动态的，执行中需根据变化持续调整
